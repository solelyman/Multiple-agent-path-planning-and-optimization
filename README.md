# Multi-USV COLREGs Cooperative Planner

This repository contains the source code for a **multi-robot cooperative path planning and optimization** framework for Unmanned Surface Vehicles (USVs), integrating contouring MPC, V-PRM topology-driven path selection, and COLREGs-compliant collision avoidance.

The main planner package is located at [`core_ws/src/multi_usv_planner/`](core_ws/src/multi_usv_planner/).

See the [planner README](core_ws/src/multi_usv_planner/README.md) for full documentation, build instructions, and usage examples.
